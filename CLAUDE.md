# Claude Code Skills

Public repository of reusable Claude Code skills and agents.

## Structure

- `skills/` — SKILL.md files following Agent Skills standard (agentskills.io)
- `agents/` — Reusable agent definitions (code-reviewer, project-setup, python-test-runner)

## Conventions

- All skills must pass `/skill-compliance-checklist` before publishing
- Include negative triggers (what the skill should NOT do)
- Licence: MIT for all public skills
- Version in SKILL.md frontmatter

## Compact Instructions

When compacting, preserve the list of modified skills and their compliance status.
